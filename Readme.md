## 二维码生成器
图一乐的用于生成二维码的C库
当前状态：不可用

### 目标
- 正确 完全符合二维码标准
- 快速 速度接近或优于现有库
- 体积小 占用存储与内存空间尽可能小

### 期望特性
- 支持二维码版本1~40，并可裁剪以减少空间占用
- 支持由外部进行内存管理，不申请释放内存
